# Five-in-a-row 技能五子棋操作说明

## 游戏简介

该游戏为特殊的五子棋单机游戏，玩家与电脑开局被随机分配黑白子，在游戏过程中双方都拥有有限次数发动7个技能的权力，率先完成五子连线的一方获胜。

## 开发原因

该游戏灵感来源于一sketch喜剧《技能五子棋》。几乎每个人在小的时候都有过跟亲人下棋耍赖的经历，这个游戏建立在传统五子棋的规则之上，加入了7个技能，为这个严谨的益智游戏增添了一些趣味。也是我上学期的processing作业。

## 运行操作

1.下载并打开processing；
2.打开文件“sketch 260109a”；
3.添加“Source”“Usage”“Modified”库；
4.选择文件栏的添加文件，添加文件夹中的所有图片以及音频文件；
5.添加完毕后，点击运行即可。
